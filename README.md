# 🧪 QA Automation Framework – Selenium + TestNG

## 📌 Project Overview

This is a **Hybrid Test Automation Framework** built using:

* Java
* Selenium WebDriver
* TestNG
* Maven

It follows **Page Object Model (POM)** and includes reusable utilities, listeners, and retry mechanisms.

---

## 🚀 Key Features

* ✅ Page Object Model (POM) Design
* ✅ Retry Analyzer (for flaky tests)
* ✅ TestNG Listeners (reporting & logging)
* ✅ Data-Driven Testing (Excel support)
* ✅ Screenshot capture on failure
* ✅ Reusable utilities (Waits, Actions, etc.)
* ✅ Structured test cases

---

## 📂 Project Structure

```
src/test/java/
│
├── pages/           → Page classes (POM)
├── testCases/       → Test scripts
├── utilities/       → Reusable helpers
└── testData/        → Test data (Excel)
```
